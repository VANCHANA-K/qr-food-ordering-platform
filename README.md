# QR Food Ordering Platform

## Prerequisites
- .NET SDK 9.x
- Git

## Clone & Run
```bash
git clone https://github.com/VANCHANA-K/qr-food-ordering-platform.git
cd qr-food-ordering-platform
dotnet build QrFoodOrdering.sln
dotnet test QrFoodOrdering.sln
dotnet run --project src/QrFoodOrdering.Api/QrFoodOrdering.Api.csproj
```

## Environment

- Development (default)
- Test (CI)

No secrets are stored in the repository.
